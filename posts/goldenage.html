My Gold Miner inspired video game is finished!

“Golden Age” is an engaging gold mining adventure game where players collect gold daily, purchase upgrades, and strive to meet the requested quota. The game features swinging grapples, intuitive movement controls, and a shop for both temporary items and permanent character upgrades. Additionally, players can enjoy a multiplayer experience by connecting controllers in the main menu. Whether using a keyboard or a gamepad, everyone can immerse themselves in this exciting quest for riches.

Golden Age can be played Here.
The code is fully public and can be found Here.

The development

I developed this game entirely using the Unity Engine, coding in C#, and creating the graphics myself. The game architecture revolves around multiple manager scripts, each responsible for a specific aspect of the game. For instance, one manager handles scene loading, another generates levels, and others track player progress, manage upgrades, control camera positioning, and adjust lighting. This modular approach ensures the code is clear and easily expandable. Each manager interacts only with the managers necessary for its function, maintaining a streamlined and efficient system.

What I learned

Creating this game taught me a lot about game development best practices, especially the importance of modular design. By segmenting responsibilities into different managers, I learned how to keep code organized and maintainable, which will be invaluable for future projects.
